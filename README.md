# Building-advertising 自述文件
*************************************
Building-advertising，简称BA，是一个广告播放器。
##特点
1.	轻巧
		大小仅有1M多
2.	清爽
		界面简洁清爽
3.	方便
		可自动更新与页面主题更换
***********************************

###
